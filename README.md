# Animexus-V2.1.06

Animexus-V2 is a modern anime discovery and tracking platform built with Next.js 15, React 19, and Tailwind CSS. It leverages the Jikan API (MyAnimeList) to provide up-to-date anime information, trending titles, and advanced search capabilities.

## Features

- **Dynamic Hero Slider**: Discover trending and newest high-rated anime titles.
- **Advanced Search**: Search for your favorite anime using a real-time hero search component.
- **Top Rated Grid**: Explore the highest-rated anime of all time.
- **Genre Explorer**: Browse anime by specific categories and genres.
- **Responsive Design**: Optimized for all devices using Tailwind CSS and Framer Motion.
- **Interactive UI**: Built with Radix UI primitives and shadcn/ui components.

## Tech Stack

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **UI Components**: [React 19](https://react.dev/), [shadcn/ui](https://ui.shadcn.com/), [Radix UI](https://www.radix-ui.com/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/), [Motion](https://motion.dev/)
- **Data Fetching**: [Jikan API](https://jikan.moe/)
- **Icons**: [Lucide React](https://lucide.dev/), [Tabler Icons](https://tabler-icons.io/)
- **Database**: [Drizzle ORM](https://orm.drizzle.team/) with LibSQL
- **Authentication**: [Better Auth](https://better-auth.com/)
- **Charts**: [Recharts](https://recharts.org/)

## Getting Started

### Prerequisites

- Node.js 20+ 
- [Bun](https://bun.sh/) (recommended) or npm/yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Perfee123/Animexus-V2.git
   ```

2. Install dependencies:
   ```bash
   bun install
   ```

3. Run the development server:
   ```bash
   bun dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

The project is optimized for deployment on [Vercel](https://vercel.com).

```bash
vercel
```
